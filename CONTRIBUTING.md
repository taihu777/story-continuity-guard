# Contributing

Thanks for helping make AI-assisted storytelling more reliable.

1. Open an issue describing the continuity failure or rule you want to address.
2. Keep changes local-first; new network dependencies require strong justification.
3. Add or update a test for every validator behavior change.
4. Run `python -m unittest discover -s tests -v` and the plugin validator before opening a pull request.

Please avoid submitting copyrighted manuscripts or private story material in public issues. Minimal synthetic fixtures are preferred.
